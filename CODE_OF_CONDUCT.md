# Contributor Covenant Code of Conduct

## 我们的承诺

为了营造一个开放且友好的环境，我们作为贡献者和维护者承诺：不论年龄、体型、身体健全与否、民族、性别认同与表达、经验水平、教育程度、社会地位、国籍、个人外貌、种族、宗教或性取向，我们项目和社区的参与者都免受骚扰。

## 我们的准则

有助于创造积极环境的行为包括：

* 使用友好和包容的语言
* 尊重不同的观点和经验
* 优雅地接受建设性批评
* 关注对社区最有利的事情
* 对其他社区成员表示同理心

参与者不应采取的行为包括：

* 使用性化的语言或图像，以及不受欢迎的性关注或性挑逗
* 捣乱/煽动/造谣行为或进行侮辱/贬损的评论，人身攻击及政治攻击
* 公开或私下的骚扰
* 未经明确授权便发布他人的个人资料，例如住址或是电子邮箱
* 其他可以被合理地认为不恰当或者违反职业操守的行为

## 我们的责任

项目维护者有责任诠释何为"可接受的行为"，并采取恰当且公平的纠正措施来应对任何不可接受的行为。

项目维护者有权利和责任去删除、编辑或拒绝与本行为准则相悖的评论（comments）、提交（commits）、代码、wiki 编辑、问题（issues）等贡献；项目维护者可暂时或永久性的禁止任何他们认为行为不当、威胁、冒犯、有害的参与者。

## 适用范围

当一个人代表该项目或其社区时，本行为准则适用于其项目空间和公共空间。代表项目或社区的示例包括：使用官方项目电子邮件地址、通过官方社交媒体账户发布内容、或在线上或线下事件中担任指定代表。项目的呈现方式可由其项目维护者进行进一步的定义及解释。

## 贯彻落实

可以通过 GitHub Issues 或联系项目维护者来举报滥用、骚扰或其他不可接受的行为。维护团队将审查和调查所有投诉，并以团队认为必要且适合实际情况的方式作出回应。项目团队有义务对事件举报者的信息保密。具体执行方针的进一步细节可能会单独发布。

未切实遵守或执行本行为准则的项目维护人员，经项目负责人或其他成员决议，可能被暂时或永久地剥夺其参与资格。
